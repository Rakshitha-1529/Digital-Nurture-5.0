**Recursion**
Recursion is a technique where a method calls itself to solve smaller instances of the same problem.

**Optimization**
To avoid repeated calculations, memoization is used. It stores previously computed values and reuses them when needed, improving efficiency.

**Time Complexity**
Recursive Approach: O(n)
Memoized Approach: O(n)     #where n is the number of years.

**Conclusion**
The recursive solution is simple and effective for financial forecasting. Using memoization further improves performance by reducing unnecessary computations.
